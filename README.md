# LeadPredictor Calculator

A simple responsive lead prediction calculator built with HTML, CSS and JavaScript.

## Features

- Calculates required customers from total revenue and average order value.
- Calculates required leads from customer count and lead response rate.
- Calculates required prospects from lead count and prospect response rate.
- Interactive sliders.
- Dynamic chart.
- Currency selector.
- Responsive dark UI inspired by the provided screenshot.

## Formulas

```text
Customers = Total Revenue / Average Order Value

Leads = Customers * 100 / Lead Response Rate

Prospects = Leads * 100 / Prospect Response Rate
```

## How to run locally

Open `index.html` in the browser.
